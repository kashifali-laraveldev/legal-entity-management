## 📌 Overview

**GRM Portal** is a full-featured enterprise-grade **Group Reporting & Legal Entity Management** platform built for finance teams, legal officers, and executive management. It centralizes the tracking of legal entities, shareholding structures, financial deliverables, audit trails, and compliance deadlines across large corporate groups.

Whether you're managing 5 or 500 legal entities across multiple jurisdictions, GRM Portal gives your organization complete visibility and control.

## ✨ Key Features

| Feature | Description |
|---|---|
| **Legal Entity Management** | Full CRUD for 50+ entities across global jurisdictions |
| **Ownership Graph** | Interactive tree-view of parent-subsidiary relationships |
| **Report Matrix** | Track 10+ deliverable types per entity per period |
| **Timeline View** | Chronological deadline tracker with live status |
| **Share Holders** | Map all ownership relationships with % and status |
| **Draft Entities** | Separate tracking for dormant/disposed/liquidating entities |
| **Finance Statistics** | Aggregate financial KPIs with charts |
| **Document Vault** | Per-entity document storage with expiry reminders |
| **Deliverable Templates** | Centralized template library with download support |
| **Period Management** | Generate and manage bi-annual reporting periods |
| **Role-Based Access** | 5 roles × granular module-level permissions |
| **User Management** | Full user lifecycle management with AD ID support |
| **Audit Trail** | Immutable system-wide activity log |
| **Email Notifications** | Scheduled alerts for deadlines, overdue, and summaries |
| **Excel / PDF Export** | Export matrices, reports, and ownership graphs |

- **Frontend:** Blade / JavaScript
- **Database:** MySQL
- **Charts:** Chart.js / D3
- **Auth:** Role-based session authentication
- **Exports:** Excel & PDF generation
